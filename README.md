# papermc.io

## Development

### Getting started

First, install the dependencies. We use pnpm as the package manager.

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
