# papermc.io

## Development

### Getting started

First, install the dependencies. We use [bun](https://bun.sh/) as the package manager.

```bash
bun install
```

Then, run the development server:

```bash
bun dev
```

Open [http://localhost:4321](http://localhost:4321) with your browser to see the result.
